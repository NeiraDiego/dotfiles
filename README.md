Uso:
- Clonar este repositorio en el home
- Instalar los programas
- usar `stow programa` por ejemplo: `stow nvim`

recomendado:
Instalar [oh-my-bash.sh](https://github.com/ohmybash/oh-my-bash)
Instalar [Nerd Fonts](https://www.nerdfonts.com/)
Para neovim:
``` bash
sudo add-apt-repository ppa:neovim-ppa/unstable
sudo apt update
sudo apt install neovim
```
